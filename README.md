# 🌐 Personal Portfolio – Yash Bisht

A modern and responsive personal portfolio website built using **HTML**, **CSS**, and **JavaScript**.  
This portfolio showcases my projects, skills, achievements, and contact information in a clean and visually appealing manner.

---

## 🚀 Features

- **Fully Responsive Design** – Works smoothly on desktop, tablet, and mobile devices.
- **Interactive UI** – Smooth animations and transitions for an engaging experience.
- **Project Showcase** – Displays my best work with links to GitHub repositories.
- **About Section** – Highlights my background, interests, and technical skills.
- **Contact Section** – Quick access to my social media and communication channels.
- **Clean & Minimal UI** – Focuses on readability and user-friendly design.

---

## 🛠️ Tech Stack

| Technology | Description |
|-----------|-------------|
| **HTML5** | Page structure & semantic layout |
| **CSS3**  | Styling, flexbox/grid, animations |
| **JavaScript** | Interactive elements & dynamic UI |
